Menu.js

Menu.js is a library for easily creating both static and dynamic menu interfaces.

Requirements

The only other library that Menu.js requires is jQuery.

Static Menus

Static menus are created by making html elements of this form: 

			<div class="menu" id="menu_name">

			    <div class="menu-item">Menu Item 1 Text</div>

			    <div class="menu-item">Menu Item 2 Text</div>

			    <div class="menu-item">Menu Item 3 Text</div>

			</div>

Menus must have unique ID attributes!

Menus must include the class "menu". Menu items must include the class "menu-item".

When the your document loads, abstract javascript models of your menus will be created. You can access them with the global_menus variable. To access your menu objects, use global_menus.getMenu("the_id_of_your_menu"). This function returns the menu object corresponding to the id you feed it!
